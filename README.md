# Expense Tracker

### Автор:
Ляпунова Алина

### Описание:
Expense Tracker — простая программа для учета личных расходов с фильтрацией по категории и дате, подсчетом суммы за период. Данные хранятся в expenses.json.

### Как использовать:
1. Запустите main.py (`python main.py`)
2. Заполняйте поля: сумма, категория, дата, нажмите «Добавить расход».
3. Используйте фильтры сверху для вывода расходов по категории или дате.
4. Общий расход за выбранный период отображается ниже таблицы.

### Примеры:
- Добавьте расходы: 400 руб., "Еда", "2024-06-12".
- Фильтруйте по категории "Транспорт" и по дате.
- Все данные сохраняются автоматически.

### Требования:
- Python 3.x
- tkinter (обычно уже в комплекте)

### Тестирование:
- Попробуйте добавить расход с отрицательной суммой или неверной датой для проверки ввода.
